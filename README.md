# Bishkek Parking AI

**Bishkek Parking AI** — дипломный проект интеллектуальной веб-системы для мониторинга, анализа, прогнозирования и рекомендации парковочных мест в городе Бишкек.

Система позволяет водителям искать парковку, получать рекомендации, а администратору — управлять парковками, зонами, пользователями, инфраструктурой рядом и ML-моделью прогнозирования загруженности.

---

## 1. Цель проекта

Цель проекта — разработать веб-систему, которая помогает:

- отслеживать текущую загруженность парковок;
- анализировать историю загрузки;
- прогнозировать будущую загруженность парковочных мест;
- рекомендовать пользователю наиболее подходящую парковку;
- учитывать инфраструктуру рядом с парковкой: вузы, школы, торговые центры, кафе, кинотеатры, офисы, рынки и жилые зоны.

---

## 2. Основная идея

Обычная система парковок показывает только текущее количество свободных мест.
